Efectos y Filtros - Tercera Parte

  El objetivo fundamental de este tema es continuar mostrando y definiendo los diversos flitros o efectos que disponemos en el Audacity. En los primeros apartados de este  tema se han visto algunos flitros y su aplicación práctica.

La finalidad de este tema es generar un mix o masterización de una canción original con nuestra propia voz, simulando un dueto con el vocalista original. Para ello tendremos que hacer uso de las propiedades de la pista (nombre, etiquetas, entre otras) y uso de herramientas de edición como dividir una pista (split).
 
Para lograr un efecto más creíble, aplicaremos diversos efectos, tanto a la vocalización original (eco) como a nuestra intervención (eco, eliminar ruido, amplificar, invertir, entre otras).
