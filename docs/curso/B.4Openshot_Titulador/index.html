Introducción

En este apartado del Editor de Video Openshot procedemos a explicar el uso del Editor de títulos. Con el uso del editor podemos generar subtítulos, mensajes, titulaciones de presentaciones en la línea de tiempo y la generación de créditos.

Una vez generado los títulos, el programa los trata como un clip más. Es decir, podemos aplicarle animaciones, efectos, transiciones y el resto de propiedades aplicadas a los clips.
