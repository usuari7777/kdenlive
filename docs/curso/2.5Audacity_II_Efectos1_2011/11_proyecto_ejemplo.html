1.1. Proyecto Ejemplo

Proyecto Explicativo

Crearemos un proyecto nuevo denomidado efectos1.aup

Importaremos una canción o fichero de audio con, al menos, un minuto y medio de duración.

Aplicaremos a este fichero una serie de efectos de sonido  para mejorar, distorsionar o eliminar aspectos del fichero original.

Una vez aplicados los efectos exportaremos el fichero a un formato estándar (mp3, flac, ogg,  wav) con el nombre de efecto1.

Reproduciremos el fichero para comprobar la diferencia entre el fichero original y el producido con nuestra edición.
