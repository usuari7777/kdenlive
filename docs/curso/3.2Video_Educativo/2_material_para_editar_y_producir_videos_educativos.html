2. Material para Editar y producir videos educativos

Recursos Necesarios

  • Ordenador: Este ordenador debe tener un procesador de altas prestaciones.

    Entre ellas podremos mencionar que disponga de Doble núcleo; Memoría RAM mínima de 2GB; Disco duro de 320 GB y con uno externo para guardar copia de los mismos, tarjeta de sonido, preferentemente preparada para sonidos envolventes (Home Cinema); tarjeta Fireware (IEE1394) para transmisión de video desde dispositivos (cámaras) habilitadas para tal fin; tarjeta capturadora de video (conocidas como tarjetas de TV/Radio) que permiten capturar los videos analógicos (VHS, mini 8, mini-DV, entre otros) y transformarlos en video digital; tarjeta gráfica de altas prestaciones (compatible linux y con un mínimo de 128 MB); grabador de DVD; dispositivos USB 2.0 y pantalla gráfica de altas prestaciones (17” pulgadas o más).

    Las características son las ideales, ello no significa que podamos realizar edición de video con equipos de inferiores prestaciones, pero el tiempo de proceso se verá limitado a dichas características.

Si no vamos a capturar video analógico o nuestra cámara de video dispone de conexiones firewire, usb u otras que permitan la transmisión directa no requerimos el uso de una tarjeta capturadora de video.

  • Programas Requeridos: Durante el desarrollo de este curso hemos mencionado diversidad de programas que podremos utilizar para la generación del material didáctico requerido utilizando video. A continuación mencionamos los más importantes:

    • Gimp: retocar imágenes estáticas

    • Audacity: editar sonido

    • Kino, Openshot, Lives, kDenlive, entre otros para editar el video

    • ManDVD, DVDStyler para generar el DVD final con títulos y capítulos

    • K3B, Brasero o GNOME BAKER para grabar en el dispositivo físico nuestro CD/DVD

Los programas que hemos enumerado pertenecen todos a Software Libre. Existe infinidad de programas de software privativo que podemos utilizar.
