3. Instalación del Kdenlive

Pasos para Instalar el Kdenlive
INSTALACIÓN DEL KDENLIVE EN NUESTRO SISTEMA
 

Disponemos de tres opciones, básicamente, para instalar nuestro programa en Ubuntu:

  • Una terminal (Aplicaciones / Accesorios / Terminal);
    • Abre una terminal y teclea sudo apt-get install kdenlive
 
  • Utilizando el Gestor de Paquetes Synaptic (Sistema / Administración / Gestión de paquetes Synaptic);
    • Busca el paquete (1), marca para su instalación (2) y aplica (3) para proceder a su instalación
 
  • Utilizando el Centro de Software de Ubuntu (Aplicaciones / Centro de Software de Ubuntu / Sonido y Video);
    • Entra en Audio y Video (A), teclea en búsqueda kdenlive (B) y, una vez mostrado la aplicación, procede a instalar (C).



Más allá - Instalación de Paquetes
