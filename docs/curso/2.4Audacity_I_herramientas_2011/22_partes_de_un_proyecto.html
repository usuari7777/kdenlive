2.2. Partes de un Proyecto

Más allá

Al guardar un proyecto se generan dos aspectos en la carpeta que lo guardas: un fichero con extensión y una carpeta con el nombre del proyecto y el sufijo data. Ejemplo: si guardarmos el proyecto como Ejemplo, nos genera el fichero Ejemplo.aup y la carpeta Ejemplo_data.

Nunca borres dicha carpeta ya que contiene las diversas partes del proyecto (audio, efectos, entre otros). Si borras dicha carpeta, al intentar abrir el proyecto nuevamente, no se abrirá y nos desplegará un mensaje que indica que la carpeta del proyecto no existe.
