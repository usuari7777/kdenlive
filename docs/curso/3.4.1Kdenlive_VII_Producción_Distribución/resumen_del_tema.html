Resumen del Tema

Reflexión del Tema

Este tema ha querido mostrar las diversas formas en las que podemos producir y distribuir nuestras ediciones de video para su reproducción.

Primero, dentro de la generación física, mostramos como generar un DVD con nuestro proyecto o con diferentes videos o clips producidos con anterioridad.

Segundo, mostramos como distribuir nuestros videos utilizando una de las herramientas de la web 2.0 de compartir videos. En este caso hemos hecho referencia a la web más conocida: Youtube.

Tercero, en otro tema de este curso se habla como capturar videos a nuestro sistema, en este apartado de producción hemos querido mostrar como podemos obtener videos de DVD's (u otros medios) ya grabados con anterioridad.

Es importante, para el desarrollo de la actividad, disponer de una conexión a intenet y de un ordenador con grabador de DVD's para que los alumnos puedan realizar las prácticas.
