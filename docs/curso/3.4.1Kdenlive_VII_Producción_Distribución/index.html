Introducción

La finalidad de este módulo es mostrar diversas funciones y utilidades que podemos aprovechar en nuestro sistema para sacar un mayor provecho a funcionalidades con el video: producir nuestros videos en soportes físicos (DVD's) y distribuir o compartir nuestros trabajos en la red (Youtube, por ejemplo).

El software libre nos brinda muchas posibilidades para realizar ambos cometidos.

En primer lugar, la producción de nuestros videos se puede realizar utilizando nuestro editor Kdenlive o una aplicación adicional (mandvd, brasero, K9copy, entre otros).

Por su parte para distribuir nuestros trabajos en el mundo virtual requerimos utilizar los servicios de un servidor de archivos. Existen muchos, como ejemplo, utilizaremos el youtube que es un sitio web en el cual los usuarios pueden subir y compartir vídeos.
