2. Generar DVD personales con MANDVD

Introducción

Una vez editados nuestros videos, normalmente, nuestra finalidad es la producir un DVD con nuestras películas o clips. Los editores, caso del Kdenlive, disponen de apartados en el menú File / Asistente para DVDs. En otros casos, generamos nuestros clips en los formatos respectivos y, posteriormente, utilizamos un programa paa generar el DVD de nuestro video. En él mismo podemos realizar menús, títulos, capítulos, etc.

Uno de estos programas es el MANDVD.  Otros programas, los quemadores o burning, disponen de apartados para la generación de DVD de vídeo (Brasero, K3b, entre otros).

 

Dentro de este apartado mostramos el ManDVd y un videotutorial resumiendo los apartados principales para la generación de nuestro DVD de video.
