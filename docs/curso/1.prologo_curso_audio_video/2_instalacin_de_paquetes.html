2. Instalación de paquetes

Introducción

Para instalar paquetes en distribuciones Ubuntu (caso del LliureX) disponemos, básicamente, de tres formas, las dos primeras en entorno gráfico y uno utilizando una terminal (comandos).

  • Utilizando el gestor de paquetes (synaptic)
  • Utilizando el Centro de Software Ubuntu
  • Utilizando una terminal. Esta opción requiere de conocimientos de comandos.

 

Si eres un usuario experto, obviamente, utilizarás una consola con el comando apt-get. Aunque, en el transcurso del curso haremos referencia a instalaciones y uso de comandos desde terminal y podrás familiarizarse con esta forma de instalar paquetes.

El resto de los usuarios puede utilizar el modo gráfico para utilizar el gestor de paquetes: Gestor de paquetes synaptic o la opción Aplicación / Centro de Software de Ubuntu.
