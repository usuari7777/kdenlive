A- Crear un podcast

Este video muestra la definición de un podcast y como registrarse en goear y subir nuestro fichero mp3 para acceder al mismo.

 


B- Podcast en Poderato

Este video muestra como podemos utilizar la web Poderato para subir nuestros podcasts. Este servidor de podcast define podcast como el conjunto de ficheros (album) de una serie y, cada uno de nuestros podcast o ficheros mp3, les denomina episodios.

C- Alojar podcast en Goear

Este video muestra los pasos para alojar nuestros ficheros de audio (podcast) en el servidor Goear.

 


D- Crear podcast, subirlo y añadirlo a un blog
