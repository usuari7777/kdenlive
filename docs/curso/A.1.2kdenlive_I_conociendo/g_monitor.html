G. Monitor

Menu Monitor

El menú monitor tiene que ver con la pantalla o monitor que disponemos en la pantalla principal y todas las funciones que se requiere. El programa nos muestra tres tipos de monitores:

  • Clip, se refiere al clip seleccionado en el árbol de proyecto;
  • Proyecto: nos muestra la reproducción de nuestro proyecto con todos los clips que hayamos añadido;
  • Grabación: nos permite capturar video desde una tarjeta capturadora de video, una webcam u otro dispositivo asociado.

Este menú incluye los Set Zone In y Set Zone Out que nos permite seleccionar parte de nuestro monitor.

 

 

El resto de las opciones no merecen mayor explicación: reproducir, rebobinar, ir a una posición determinada, rebobinar x fotogramas, avanzar, parar, entre otras.

En la zona derecha de la aplicación nos muestra el monitor y las acciones del menú que estemos aplicando.

 

CAPTURAR VIDEO DESDE MONITOR: Nos ubicamos en la pestaña del Monitor de grabación. Seleccionamos el dispositivo desde el que vamos a realizar la grabación (fireware, captura de pantalla o grabar escritorio y Video4Linux). En este caso vamos a utilizar Video4Linux (utilizado para las capturas de video, en este caso una webcam conectada a través del puerto USB).

 

 

Para inicializar la captura primero debemos hacer play (>) para que se inicie el dispositivo.

 

 

Una vez incializado (muestra la emisión de nuestra webcam) ya podemos iniciar la grabación para ello debemos indicar otras opciones:

 

 

  Se nos despliega el submenú y procedemos a grabar. Al finalizar pulsamos detener. La grabación se realiza en la carpeta y con el nombre establecido en el menú settings / configurar Kdenlive / entorno /carpetas determinadas / carpeta de captura.
